# 作业管理系统前端

一个使用 Vue 3 + Element Plus 构建的现代化作业管理系统前端应用。

## 功能特性

- 用户登录和注册
- 作业列表和详情查看
- 学生作业提交
- 教师作业批改
- 优秀作业展示
- 个人资料管理
- 响应式设计

## 技术栈

- Vue 3
- Element Plus
- Vue Router 4
- Pinia
- Axios
- Vite
- TypeScript

## 快速开始

pnpm install
pnpm dev

## 环境配置

创建 .env.local 文件：
VITE_API_URL=http://your-api-server:8080/api
